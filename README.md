# Superstore Sales Dashboard

## Project Overview

I created this Tableau dashboard to analyze sales and profit performance using the Superstore dataset. The dashboard helps understand how sales vary across different months, regions, and product categories.

The goal of this project was to practice data visualization and turn raw sales data into meaningful business insights.

## Tools Used

- Tableau
- CSV Dataset
- Data Visualization

## Dashboard Features

### Monthly Sales Trend
This chart shows how sales changed throughout the year and helps identify high-performing months.

### Sales by Region
This visualization compares sales across different regions and highlights the strongest and weakest performing areas.

### Sales and Profit by Sub-Category
This chart provides a detailed view of product performance by comparing sales and profit for each sub-category.

## Key Findings

- The West region generated the highest sales among all regions.
- The South region had the lowest sales performance.
- Sales increased significantly during the last few months of the year.
- Phones and Chairs were among the best-selling product sub-categories.
- Some categories generated high sales but relatively lower profits.
